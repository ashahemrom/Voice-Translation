🚀 How to Use the Full Application

  1. Install Flask and SocketIO: pip install flask flask-socketio python-socketio
  2. Run the main application: python main.py
  3. Choose option 3: "🌐 Web Interface (Browser-based UI)"
  4. Open your browser to http://localhost:5000
  5. Select patient language and start voice conversations!
